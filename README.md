# DevHub

DevHub is a personal dashboard built using HTML, CSS, and JavaScript. It brings together a few everyday utilities in one place to help stay organized and productive while serving as a hands-on project to strengthen my frontend development skills.

## Features

- 👋 Dynamic greeting based on the current time
- ⏰ Live digital clock
- 📅 Current date
- 🎯 Daily goals tracker
- 📊 Productivity tracker with completion statistics and progress bar
- 🌤️ Current weather for Surat
- 💬 Random inspirational quote generator
- 🌄 Daily landscape image
- 🔗 Quick access to frequently used websites
- 🌙 Light / Dark mode toggle

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Local Storage
- Flexbox & CSS Grid

---

## APIs Used

- **OpenWeatherMap API** – Current weather
- **DummyJSON Quotes API** – Random quotes
- **Lorem Picsum** – Landscape images
